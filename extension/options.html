Microsoft Edge TTS Free

Natural-sounding neural voices from Microsoft — completely free, no API key required. When selected, replaces browser TTS for article reading.

AI Podcast

Converts any article into an NPR-style audio podcast with ambient music and multiple voices. Requires an Anthropic API key.

Stored locally on this device only. Get a key at console.anthropic.com — the 🎙 button in the player will prompt you if not set.

ElevenLabs (High-Quality TTS)

Use ElevenLabs for natural-sounding voices when reading articles and for the 👥 Discussion Podcast. Free tier gives ~10,000 chars/month (~1,600 words). Get a key at elevenlabs.io.

When an ElevenLabs key is set, it will be used for article reading and discussion podcasts. Falls back to browser voices if not set.
